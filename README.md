### Installation & Setup

1. **Clone Repository**
```bash
git clone
cd folder
npm install
```

1. **Install Dependencies:**
```bash
npm install
```

2. **Start the Server:**
```bash
node server.js
```

The server will run on `http://localhost:1234`
